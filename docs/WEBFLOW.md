# WEBFLOW — Путь пользователя по сайту sitechist.ru

## Обзор

Сайт — воронка, ведущая к одному действию: **переход в Telegram-бот @WebAuditRuBot для бесплатного аудита**. Каждый блок усиливает мотивацию и снижает сопротивление. Вторичная цель — сбор лидов через форму заявки.

---

## Точки входа на сайт

```
Прямой заход (sitechist.ru)
Яндекс/Google (SEO)
UTM-ссылки (реклама)
Telegram-канал / рассылка
AI-консультант (виджет на сайте)
```

UTM-метки собираются автоматически из URL и передаются в бота через deep link.

## Параллельные элементы (fixed/overlay)

### StickyCta (mobile only, z-1000)
```
Появляется после прокрутки блока «Штурм» (#roadmap)
┌─────────────────────────────────┐
│ [ОСТАВИТЬ ЗАЯВКУ] [📩 TG-бот]  │
└─────────────────────────────────┘
• «Оставить заявку» → Lead Modal
• TG-иконка → ?start=free_audit__sticky_cta
• safe-area-inset для iPhone
```

### AI-консультант (VoiceWidget, z-9998)
```
🎙 Кнопка — fixed bottom-right
   Появляется после блока «Штурм»
   Пульсирующее кольцо + attention pulse

Панель чата:
   • Чипы быстрых вопросов:
     🛡️ Бесплатный аудит · Риски · Штрафы · Тарифы
   • Текстовый ввод + кнопка отправки
   • Кнопка микрофона (Gemini Live)
   • Промпт загружается из /ai-consultant-prompt.txt

Действия AI:
   • redirectToBot() → ?start=free_audit__ai_consultant
   • submitLead(data) → POST /api/ai-lead → TG + Sheets
```

### Cookie Banner (z-200)
```
Появляется через 20 сек
Автоскрытие через 20 сек после появления
Кнопка «Хорошо»
localStorage: cookie_accepted (24h TTL)
Позиция: bottom-20 (над StickyCta)
```

## Техническая карта

```
                    ┌──────────────┐
                    │  sitechist.ru │
                    │  (Vite SPA)   │
                    └──────┬───────┘
                           │
              ┌────────────┼────────────┐
              │            │            │
         ┌────▼────┐  ┌────▼────┐  ┌───▼────┐
         │ Express  │  │ Resend  │  │ Gemini │
         │ :3002    │  │ (email) │  │ Live   │
         └────┬────┘  └─────────┘  └────────┘
              │
     ┌────────┼────────┐
     │        │        │
┌────▼───┐ ┌──▼──┐ ┌───▼───────┐
│Telegram│ │Sheets│ │   Nginx   │
│Bot API │ │(CRM) │ │ :443/80   │
└────────┘ └──────┘ │ SSL, gzip │
                    │ /api → :3002│
                    │ /ws → :3002 │
                    └─────────────┘
```

**Nginx locations:**
- `/` → static files (`/var/www/sitechist.ru/`)
- `/api/` → Express proxy (`:3002`)
- `/gemini-api/` → Gemini HTTP proxy
- `/ws-gemini` → Gemini WebSocket proxy

**PM2 processes:**
- `gemini-proxy-sitechist` — Gemini proxy (порт 3001)
- `sitechist-api` — Express API (порт 3002)
